Privacy Policy

Product: Valentte Ad Automator

Effective date: March 11, 2026

1. What This Service Does

Valentte Ad Automator helps create and manage Meta ads in bulk using customer-provided spreadsheet data and media links.

2. Data We Process

We may process the following data when running ad automation:

  • Ad configuration data you provide (ad set IDs, ad names, text, headlines, URLs, status fields).
  • Creative asset references (image/video links and file paths).
  • Platform identifiers (Meta ad account ID, Page ID, Instagram user ID).
  • Operational metadata (timestamps, run status, error messages, created ad/creative IDs).

3. What We Do Not Intend To Collect

We do not intentionally collect sensitive personal data through this tool, and we do not request user passwords.

4. How Data Is Used

Data is used only to:

  • Create and manage ads and creatives via Meta APIs.
  • Read/write workflow status in connected spreadsheets.
  • Troubleshoot and monitor job execution.

5. Third-Party Services

This workflow interacts with third-party services you authorize, including:

  • Meta Platforms (Marketing API)
  • Google (Sheets API)
  • Dropbox (media retrieval)
  • GitHub Actions (automation runtime)

Each provider processes data under its own terms and privacy policy.

6. Data Retention

Data retention is minimized to operational needs. Job data may remain in logs and spreadsheets as required for campaign operations and troubleshooting. You can request deletion as described below.

7. Security

Access tokens and credentials are stored in environment secrets where available. Access is limited to authorized operators and systems.

8. Data Deletion Requests

To request deletion, follow the instructions at: User Data Deletion.
